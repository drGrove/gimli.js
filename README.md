# Gimli.js

An implementation of [Gimli][gimli] in typescript with compilation to javascript in CommonJS.


## Currently implented functions

* Core Permutation
* Hash
* Cipher

## TODO

* PRNG

Based on the [zig][zig] implementation by [daurnimator][daurnimator].
